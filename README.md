Завдання 1
-
Відкрити завдання 2 (3 уроку) та продовжити розробку інтернет магазину.

Наразі ми маємо лише "сухі" дані, вони не можуть застосовуватись напряму з бази, їх потрібно для початку обробити. Ви можете використовувати весь створений вами функціонал з інтерфейсів ICategoryRepository, IKeyParamsRepository, IProductRepository та інші (це і є ваші методи CRUD з бази).

Відкрийте бібліотеку BLL папку Services та на прикладі класу UserServices :

* Реалізуйте інтерфейси класів всередині (щоб не залишилось NotImplementedException і були наявні усі методи).
* Уважно розгляньте метод ProductsByWord у класі ProductServices. Напишіть запит, котрий буде отримувати список товарів за назвою продукту
* Уважно розгляньте метод GetProductsByCategoryAndPrice у класі ProductServices. Напишіть запит, котрий буде отримувати увесь список продуктів за їх категорією та ціною у певному проміжку
* Уважно розгляньте метод GetProductsByCategoryAndKeyWordsWithPrice у класі ProductServices. Напишіть запит, котрий буде додатково формувати ще по ключовим словам

Після реалізації зазначених методів має працювати пошуковик, а також відображатися обраний продукт

Завдання 2
-
Відкрити завдання 1 (цього уроку).

Відкрийте бібліотеку BLL папку Services

Розгляньте метод GetCategoryInfoByName у класі CategoryServices. У ньому потрібно повернути інформацію по категорії, тобто сформувати значення:

1) Максимальної ціни
2) Мінімальної ціни
3) Категорії
4) Ключові слова до відповідної категорії у форматі Dictionary

Ваша мета сформувати Dictionary де ключем виступатиме Header з вашого класу Word, а масивом стануть усі значення KeyWord по значенню Header

Вказати всі необхідні значення для формування CategoryInfo
Внести в базу нову категорію, додати товар та вказати ключові слова (назва категорії лише латиницею та без пробілів)
![Image from hw](https://lms.cbs.com.ua/pluginfile.php/25432/mod_assign/intro/screen.png)
Якщо все правильно ви матимете на сторінці категорії виборку з вашими ключовими словами та зможете вибирати продукцію за ними.
